Dataset Description:
The dataset used in this assignment is the Telco Customer Churn dataset obtained from Kaggle. 
It is a tabular dataset containing customer demographic information, services subscribed, and account details.

Machine Learning Problem:
This problem is a supervised machine learning classification task.
The goal is to predict whether a customer will churn (leave the service) or not based on the available features.
The target variable in this dataset is the "Churn" column.
